¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que permite agregar interactividad a las páginas web. Es uno de los tres pilares del desarrollo web junto con HTML y CSS.

¿Para qué sirve?

  • Validar formularios
  • Crear animaciones
  • Manipular el DOM
  • Desarrollar aplicaciones web
  • Crear videojuegos
